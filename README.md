# Network Port Scanner

## Description
A Python-based Network Port Scanner that scans common ports on a target IP address or hostname and reports whether each port is open or closed.

## Features
- Scan common ports
- Supports IP Address and Hostname
- Detects Open/Closed Ports
- Displays Service Names
- Fast TCP Port Scanning

## Technologies Used
- Python
- socket
